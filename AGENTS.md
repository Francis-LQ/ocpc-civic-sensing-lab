# AGENTS.md

## Repository Role

This repository is the working knowledge base and product-planning workspace for OCPC / Civic Sensing Lab, including source literature, project source documents, pitch materials, and startup/product strategy documents.

## Working Rules

- Preserve original source materials. Do not rewrite or delete PDFs, DOCX, PPTX, or archived reference files unless the user explicitly asks.
- Prefer creating derived notes, summaries, schemas, plans, and structured outputs instead of mutating source documents.
- Keep Chinese project language natural and precise. Use English terms where they are established domain vocabulary, such as STEAM, OCPC, Civic Sensing Lab, AI for EDA, and open-source hardware.
- When editing Markdown, keep headings stable and avoid large unrelated rewrites.
- Do not commit temporary office lock files, Python caches, generated logs, or local environment folders.

## Project Context

- Front-stage brand: 共感城市实验室 / Civic Sensing Lab / CSL.
- Back-stage framework: OCPC / Open Civic Project Commons / 开放公民项目公地.
- Current product direction: literacy education, public projects, open civic knowledge commons, and adjacent AI/EDA research-product exploration.

## Verification

- For document edits, review the changed sections directly.
- For code or scripts, run focused tests or a minimal smoke test before committing.
- For market, policy, or current company claims, verify with up-to-date sources.
