# CSL 7 天共创挑战行动画布：Vibe Coding 身心边界感知

## 基本信息

- 活动名称：Vibe Coding 身心边界感知 7 天挑战
- 发起者公开署名或账号：Francis-LQ
- 发起者角色：项目维护者 / 社区发起者
- 目标参与者：18 岁及以上高校编程学习者、学生技术社团、开源社群成员
- 是否涉及 14–17 岁参与者：首轮不涉及
- GitHub issue：https://github.com/Francis-LQ/ocpc-civic-sensing-lab/issues/13

## 公共议题

- 我们关注什么身边问题：无边界编码、开放式任务悬置、停工困难和休息切换困难。
- 为什么它值得共同观察：它影响学习节奏、协作质量和开源项目可持续维护。
- 这个挑战不会回答什么问题：不判断焦虑、失眠或心理健康诊断，不评估疗效。

## 参与任务

- 每位参与者完成 7 天轻量记录。
- 最低完成标准：至少 5 天记录，且完成一次 Day 7 复盘。
- 可选增强任务：贡献一个任务拆解 prompt、停工清单或复盘模板。
- 不需要的设备或能力：不需要可穿戴设备、心理量表、专业咨询资质或公开个人经历。
