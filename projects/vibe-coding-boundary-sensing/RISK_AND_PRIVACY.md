# 风险与隐私清单 / Risk and Privacy Checklist

## 当前草案状态

- [x] 草案包不包含真实参与者数据。
- [x] 草案包不包含原始日记、心理自述、联系方式、身份映射、音频、图片或聊天记录。
- [x] 草案包不包含诊断、治疗、心理咨询或健康评估承诺。
- [x] 首轮默认面向 18 岁及以上参与者。
- [ ] 真实活动前完成活动说明、同意、撤回路径和风险升级流程审阅。
- [ ] 公开 release candidate 已完成授权、撤回和敏感内容审核。

## 心理健康边界

- 本项目只提供学习节奏与边界观察模板，不提供心理咨询、治疗、危机干预或诊断。
- 不使用“治愈”“疗效”“干预成功率”等表述作为公开成果指标。
- 不收集临床量表、诊断史、用药史、自伤细节或可识别危机叙述。
- 如参与者表达严重痛苦、自伤风险或需要专业帮助，发起者应停止公开处理，并建议其联系学校心理支持、当地紧急服务或可信任的专业人员。

## 发布前必须确认

- 公开摘要只使用随机编号或获授权别名。
- 公开文字已删除姓名、联系方式、学校班级、精确时间线、个人经历细节和第三方信息。
- 原始日记、聊天记录、音频、图片和学生作品不进入 GitHub。
- 撤回请求可映射到公开资产并在 7 天内处理。
