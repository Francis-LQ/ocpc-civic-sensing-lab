# Vibe Coding 身心边界感知 Rubric

| 维度 | 说明 | 分值 |
| --- | --- | ---: |
| 任务闭环 | 能把开放式编码任务拆成当日可完成单元 | 25 |
| 停工边界 | 能设置停工红线并把未完成事项移入明日停车场 | 25 |
| 恢复行动 | 能完成离开电脑、整理环境或短时恢复行动 | 20 |
| 公共表达 | 能用去标识化摘要说明方法和限制 | 20 |
| 隐私安全 | 未公开原始日记、心理自述、联系方式、精确时间线或未授权素材 | 10 |

有效 release candidate 需要达到 60 分以上，且隐私安全维度不得为 0。
