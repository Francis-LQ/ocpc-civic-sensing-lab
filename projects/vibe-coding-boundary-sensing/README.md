# Vibe Coding 身心边界感知 / Vibe Coding Boundary Sensing

这是一个 CSL 社区项目草案，背景来自本地 `Ref/` 中的高校服务学习项目方案。公开项目包只提炼“无边界编码、任务悬置、休息切换困难”这一公共议题，不重新发布原文，不公开个人经历细节或敏感心理描述。

## 定位

本项目是 7 天社区共创挑战，不是医疗、心理咨询、心理治疗、临床筛查或健康评估。它帮助高校编程学习者用低门槛方式观察自己的编码节奏、停工边界和恢复行动，并把可复用方法沉淀为 OCPC 草案包。

## 公共议题

Vibe Coding 和开源学习常常鼓励即时试错、沉浸迭代和快速反馈，但如果缺少任务闭环、停工红线和休息切换，学习者可能出现无边界工作、反复挂念未完成任务、错过日常安排或休息质量下降等问题。

本项目只讨论学习与项目管理层面的边界设计，不诊断焦虑、失眠或其他心理健康状态。

## 默认活动

- 活动模型：7 天小挑战。
- 目标参与者：默认 18 岁及以上高校编程学习者、学生技术社团或开源社群成员。
- 首轮规模：建议 5–10 人邀请制试跑。
- 公开成果：协议、Rubric、去标识化聚合摘要、复盘和复用建议。

## 公开边界

- 不公开姓名、联系方式、学校班级、精确时间线、原始日记、心理自述、音频、图片、聊天记录或学生作品。
- 不把个人困扰写成可识别案例。
- 不把活动结果写成健康、安全、治疗或机构绩效结论。
- 如果参与者出现严重痛苦、自伤风险或需要专业帮助，活动应立即停止相关公开处理，并建议其联系学校心理支持、当地紧急服务或可信任的专业人员。
