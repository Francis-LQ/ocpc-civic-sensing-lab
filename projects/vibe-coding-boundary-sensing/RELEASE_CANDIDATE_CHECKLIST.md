# Release Candidate 检查清单

## 参与者级检查

- [ ] 使用随机编号或获授权别名。
- [ ] 未公开原始日记、心理自述、聊天记录、音频或图片。
- [ ] 未公开姓名、联系方式、学校班级、宿舍地址或精确时间线。
- [ ] 未使用诊断、治疗、疗效、康复等医疗或心理咨询承诺。
- [ ] 可撤回路径已告知并记录。

## 项目级检查

- [ ] 聚合摘要不包含个人逐日记录。
- [ ] 所有公开资产能映射到撤回日志。
- [ ] `ocpc-project.json` 的公开资产列表与实际文件一致。
- [ ] `publication_review.status` 仅在全部通过后改为 `approved`。
- [ ] validate、render、单元测试和 GitHub Actions 均通过。
