# Release Candidate 检查清单

## 参与者级检查

- [ ] 已完成至少 6 条观察。
- [ ] 已覆盖微气候与声音两类字段。
- [ ] 使用随机编号或获授权别名。
- [ ] 文字反思已删除姓名、联系方式、学校班级、住址和精确位置。
- [ ] 可选个人页已单独授权；未成年人具备监护确认。
- [ ] 可选音频已有私有编号、审核状态和删除日期。

## 项目级检查

- [ ] 聚合数据不包含个人层面轨迹。
- [ ] 所有公开资产能映射到撤回日志。
- [ ] `ocpc-project.json` 的公开资产列表与实际文件一致。
- [ ] `publication_review.status` 仅在全部通过后改为 `approved`。
- [ ] validate、render、单元测试和 GitHub Actions 均通过。
