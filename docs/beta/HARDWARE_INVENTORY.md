# 校园感知 Beta 参考设备盘点

首轮不向参与者寄送硬件。维护者准备 2–3 套参考设备用于直播演示、协议校准和
后续课程复用。优先使用现有库存，缺口采购控制在人民币 500 元以内。

| 组件 | 建议数量 | 现有数量 | 缺口 | 备注 |
| --- | ---: | ---: | ---: | --- |
| ESP32 开发板 | 2–3 | 待盘点 | 待填写 | 保留 Wi-Fi 扩展空间 |
| BME280 温湿度传感器 | 2–3 | 待盘点 | 待填写 | 微气候演示 |
| BH1750 光照传感器 | 2–3 | 待盘点 | 待填写 | 光照演示 |
| 面包板 | 2–3 | 待盘点 | 待填写 | 低压连接 |
| 杜邦线 | 2–3 组 | 待盘点 | 待填写 | 低压连接 |
| USB 线 | 2–3 | 待盘点 | 待填写 | 供电与调试 |

## 暂不采购

- 麦克风模块：首轮声音数据使用手机手工记录和自愿短音频。
- CO2、PM 传感器：留到课程闭环验证后再评估。
