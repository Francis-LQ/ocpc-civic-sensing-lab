# 校园感知 Beta：14 天课程结构

## 对象与目标

- 邀请制免费内测，招募 10–20 位国内跨区域、14 岁及以上参与者。
- 验证微气候与声景并行观察、异步支持、公开发布和撤回流程。
- 不把免费 Beta 写成付费验证，不承诺完整平台功能。

## 课程结构

| 时间 | 环节 | 产出 |
| --- | --- | --- |
| Day 1 | 直播启动与安全说明 | 观察计划、模糊分区、同意记录 |
| Day 2–10 | 异步观察与答疑 | 每人至少 `3 个模糊分区 × 2 个时段` |
| Day 11 | 数据整理与公开发布诊所 | 汇总表、匿名展示草稿 |
| Day 12–14 | 复盘与人工审核 | release candidate、反馈问卷 |

## 声音边界

- 手机 dB 读数只用于同一设备内相对比较，不用于跨设备绝对结论。
- 自愿音频每人最多 3 段，每段最多 10 秒；禁止可辨识对话。
- 未成年人原音频首轮不公开。
- 成年人音频仅在单独授权和人工审核后允许选段公开。

## 公开成果

- 一个去标识化聚合项目页。
- 可选自愿个人页。14–17 岁参与者仅允许别名、去标识化图表和文字反思，
  并需参与者与监护人双重确认。
- 公开成果不得包含姓名、联系方式、精确位置、个人轨迹或未审阅素材。

## 验收指标

| 指标 | 门槛 |
| --- | ---: |
| onboarding 完成人数 | ≥ 10 |
| 两类采样完成率 | ≥ 70% |
| 有效 release candidate 生成率 | ≥ 60% |
| 安全与隐私事故 | 0 |
| 原始音频具备删除日期和审核状态 | 100% |
