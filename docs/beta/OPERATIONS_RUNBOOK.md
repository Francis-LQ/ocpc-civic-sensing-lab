# 校园感知 Beta 运营手册

> 适用期：2026-06-04 至 2026-07-04。本文是执行门禁，不构成法律意见。

## 目标

- 完成一次 14 天邀请制免费 Beta，不测试付费，不公开招募。
- 至少 10 人完成 onboarding，至少 70% 完成微气候与声音两类采样。
- 至少 60% 生成有效 release candidate，安全与隐私事故为 0。
- 公开一个去标识化聚合项目页、复盘报告和 `v0.2.1` 维护 release。

## 每日循环

1. 从飞书导出当日新增报名、上传和反馈记录。
2. 在本地加密目录更新私有主账本，不把真实身份、联系方式、授权记录或待审音频放入 GitHub。
3. 将参与者进度同步到 `participant-index.csv`，只使用随机编号。
4. 标记需要人工处理的风险：未成年人授权、音频疑似对话、精确位置、撤回请求、异常情绪或设备问题。
5. 只把审核后的聚合数据、匿名文本和公开资产复制到 `projects/campus-sensing-beta-2026-06/`。

## 撤回流程

1. 收到撤回请求当天在 `withdrawal-log.csv` 记录请求时间、范围和随机编号。
2. 暂停该参与者新增公开处理，检查公开资产映射。
3. 7 天内删除或替换相关公开资产，重新运行 validate/render 并重新部署 Pages。
4. 在私有主账本记录完成时间、重新部署链接和处理人。

## 发布流程

1. 将 `projects/campus-sensing-beta-2026-06/ocpc-project.json` 中的公开资产列表更新为真实审核后资产。
2. 只有在授权、撤回映射和音频审核全部完成后，才把 `publication_review.status` 改为 `approved`。
3. 运行：

```bash
python scripts/ocpc_toolkit.py validate projects/campus-sensing-beta-2026-06
python scripts/ocpc_toolkit.py render projects/campus-sensing-beta-2026-06 --output .tmp-site
python -m unittest discover -s tests -v
```

4. 人工检查 `.tmp-site` 中没有姓名、联系方式、精确位置、个人轨迹、未审音频或未授权图片。
5. 发布 `v0.2.1` 时只记录真实修复、公开边界和复盘结果，不写成付费验证。
