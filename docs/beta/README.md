# 校园感知 Beta / Campus Sensing Beta

本目录定义首轮 14 天邀请制免费 Beta 的公开课程边界。活动面向国内跨区域、
14 岁及以上参与者，目标是验证校园微气候与声音观察的课程闭环、工具链和安全
发布流程。

## 当前范围

- 微气候与声音并重，不加入 CO2、PM 或网页麦克风采集。
- 基础轨不寄送硬件；参考设备仅用于演示与协议校准。
- 飞书作为收集入口，本地加密目录作为唯一私有主账本。
- GitHub 只接收人工审阅后的聚合数据、协议、Rubric 和自愿展示页。
