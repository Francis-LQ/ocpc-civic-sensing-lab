# CSL 社区活动公开与私有边界

> 本文是社区活动的安全边界说明，不构成法律意见。涉及未成年人、个人信息或机构合作时，应按当地要求完成必要审阅。

## 可以公开

- 活动说明、观察协议、Rubric 和复盘。
- 去标识化聚合摘要。
- 经授权、经审阅、无可识别个人信息的文字反思。
- 不包含精确位置的模糊分区图表。
- 发起者或贡献者自愿公开的署名或别名。

## 默认私有

- 报名表、联系方式、身份映射和授权记录。
- 原始观察记录、原始音频、原始图片和学生作品。
- 未审文字、可识别对话、精确位置、个人轨迹和撤回日志。
- 任何 API Key、账号凭据、内部群聊、未授权第三方资料。

## 未成年人

- 涉及 14–17 岁参与者时，必须执行监护确认。
- 未成年人原音频不公开。
- 未成年人个人页不是默认成果，只能使用别名、去标识化图表和审阅文字。
- 不公开学校班级、家庭信息、联系方式、面部图像或可识别轨迹。

## 音频、图片和作品

- 默认不公开音频、图片或学生作品。
- 成年人音频如需公开，必须单独授权、人工审核，且不得包含可辨识对话。
- 学生作品如需公开，必须确认权利人或监护人授权，并保留撤回路径。

## 位置与数据解释

- 使用模糊分区，例如入口区、学习区、休息区。
- 不公开精确地址、楼层房间号、轨迹或打卡时间线。
- 手机 dB 读数只用于同一设备内相对比较。
- 不把活动结果写成健康、安全、政策或机构绩效结论。

## 撤回

- 公开前记录公开资产与参与者随机编号的映射。
- 收到撤回请求后，暂停新增公开处理。
- 7 天内删除或替换相关公开资产，并记录重新部署时间。
