# 示例活动：校园遮阴与声景 7 天挑战

> 这是一个无真实数据示例，用于展示社区发起者如何填写活动结构。

## 一句话

用 7 天时间观察校园或社区中的遮阴、热舒适和声景体验，整理成可复用的公共空间观察模板。

## 行动画布摘要

- 活动名称：校园遮阴与声景 7 天挑战
- 发起者角色：学生社团或教师
- 目标参与者：14 岁及以上学生、教师或社区成员
- 公共议题：哪些公共空间在不同时间更适合停留、学习或休息
- 最低任务：每人选择 2 个模糊分区，完成 2 个时段观察
- 公开成果：活动说明、观察协议、Rubric、去标识化聚合摘要、复盘
- 私有资料：报名表、联系方式、授权记录、未审照片或音频、撤回日志

## 7 天节奏

| 日期 | 任务 | 产出 |
| --- | --- | --- |
| Day 0 | 提交 Community Activity issue | 活动提案 |
| Day 1 | 说明观察任务和公开边界 | 参与说明 |
| Day 2 | 观察入口区和休息区 | 私有观察记录 |
| Day 3 | 观察学习区和活动区 | 私有观察记录 |
| Day 4 | 补充主观热舒适和声景评分 | 私有观察记录 |
| Day 5 | 整理匿名摘要 | release candidate |
| Day 6 | 安全审阅 | 发布检查清单 |
| Day 7 | 复盘 | 复盘摘要 |

## 示例推广文案

我们正在发起一次 CSL 7 天共创挑战，主题是“校园遮阴与声景”。你可以选择身边两个公共空间，记录不同时间的热舒适、声音感受和简短观察。无需专业设备，不公开个人信息，也不做排名。最终我们会把匿名摘要和方法沉淀为可复用的开放项目草案。

## 示例 issue 标题

`[Community Activity] 校园遮阴与声景 7 天挑战`

## 不进入公开包的内容

- 学生实名、联系方式、班级。
- 精确位置、个人轨迹、原始照片和原始音频。
- 未经授权的学生作品或第三方素材。
