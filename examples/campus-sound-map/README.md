# 校园声音地图 / Campus Sound Map

这是一个用于展示 OCPC 项目包结构的合成示例，不代表已经开展真实校园数据采集。

## 公共议题 / Civic Issue

校园中哪些公共区域在不同时段更需要关注声音环境？如何用数据帮助学生讨论公共空间的共同使用？

## 方法 / Method

示例将校园抽象为入口区、学习区和活动区，只提供人工生成的分区级声音强度摘要。真实项目应先完成参与者授权、安全评估和数据最小化设计。

## 公开成果 / Public Artifacts

- `artifacts/sample-summary.csv`：合成的分区级声音强度摘要。
- `RISK_AND_PRIVACY.md`：本示例的数据发布边界。
- `CONTRIBUTORS.md`：公开贡献记录示例。

## 复用指南 / Reuse Guide

复用时不得记录学生姓名、联系方式、面部图像、个人轨迹或精确坐标。优先发布分区级、时段级聚合数据。
