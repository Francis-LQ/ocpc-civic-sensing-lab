# AGENTS.md

## Directory Role

`源文档` contains original and classified literature/source materials used to support OCPC / Civic Sensing Lab research.

## Working Rules

- Treat files here as source evidence. Do not modify original documents unless explicitly requested.
- Create summaries, extraction tables, and reading notes as separate Markdown files.
- Maintain classification folders and avoid moving files across categories without user confirmation.
- When citing source materials, record filename, page number if available, and the extracted claim.
